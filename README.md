# YukonSYS-Standard-Definitions
A collection of language definitions + syntax standards for programming, analytics, automation services, and systems architectures

## Repository Creation Policy

All new YukonSYS repositories MUST be created from this repository as the
GitHub template baseline. See
`.structs/opersys/git/repository-template-policy.md` for the approved creation,
retrofit, exception, and verification process.

## GitHub Repository Standards

Reusable GitHub issue forms, pull request templates, label taxonomy, and commit
message guidance live under:

```text
.github/ISSUE_TEMPLATE/
.github/PULL_REQUEST_TEMPLATE/
.structs/opersys/git/
```

Start with `.structs/opersys/git/github-templates-compliance.md` when adapting
these standards into another YukonSYS repository.
